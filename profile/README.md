## Explore Repositories

- [platform-web3-protocol](https://github.com/ReineiraOS/platform-web3-protocol): core protocol — FHE-encrypted escrow engine, pluggable insurance, orchestration, confidential tokens, and TypeScript SDK.
- [reineira-code](https://github.com/ReineiraOS/reineira-code): AI-assisted plugin development — describe what you want, Claude Code generates the Solidity contract, tests, and deployment.
- [reineira-atlas](https://github.com/ReineiraOS/reineira-atlas): startup operating system — AI agents for strategy, growth, compliance, tokenomics, and investor readiness.
- [platform-modules](https://github.com/ReineiraOS/platform-modules): plug-and-play application starters — backend API, platform app, and shareable payment link.

## ReineiraOS Tools

- [ReineiraOS SDK](https://www.npmjs.com/package/@reineira-os/sdk): TypeScript SDK for escrow creation, insurance coverage, and cross-chain settlement.
- [Condition Resolvers](https://reineira.xyz/docs/develop/condition-plugins): pluggable contracts that control when escrows release funds — zkTLS proofs, oracle thresholds, multi-sig, time locks.
- [Insurance Policies](https://reineira.xyz/docs/develop/insurance-policies): FHE-encrypted risk evaluation and dispute resolution — chargeback protection, trade credit, parametric coverage.
