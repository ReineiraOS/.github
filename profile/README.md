## Build with ReineiraOS

- [**ReineiraOS SDK**](https://www.npmjs.com/package/@reineira-os/sdk) — TypeScript SDK for escrow creation, recourse coverage, and cross-chain settlement.
- [**Condition Resolvers**](https://docs.reineira.xyz/build/condition-resolvers) — pluggable contracts that control when escrows release funds: zkTLS proofs, oracle thresholds, multi-sig, and time locks.
- [**Recourse Policies**](https://docs.reineira.xyz/build/underwriter-policies) — FHE-encrypted risk evaluation and dispute resolution: chargeback protection, trade credit, and parametric coverage.
